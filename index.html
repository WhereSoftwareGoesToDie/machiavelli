Machiavelli - Anchor

Before all else, be graphed

Machiavelli is a Ruby on Rails application that can be used to query different backend datastores, and then graph any arbitrarily defined metric from that source. It allows cross-source graphing, multiple visualisation styles, and sharing a graph is as easy as copying the URL. Backends already written include flat CSV files, simple JSON feeds, and time-series datastores such as Graphite, RRD, New Relic, and Vaultaire.
